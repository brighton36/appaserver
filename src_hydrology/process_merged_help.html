Output Merged Datasets

This process compares the measurements from multiple datatypes with each other.

  • Begin and End Dates:
    • Leave both blank will report the entire period of record.
    • Enter a full Begin Date and End Date will report on that date range.
    • Enter only the year in Begin Date will report from January 1.
    • Enter only the year in End Date will report to December 31.
    • Enter only the year and month in Begin Date will report from the first day of that month.
    • Enter only the year and month in End Date will report to the last day of that month.
    • Enter a full Begin Date and leave the End Date blank will report on the single day.
    • Enter a only the year in Begin Date and leave the End Date blank will report on the entire year.
    • Enter a only the year and month in Begin Date and leave the End Date blank will report on the entire month.
    • Leave blank the Begin Date but populate the End Date will report from the beginning of record to the End Date.
  • Aggregate Level: Optionally select an aggregate level to compare sums (if quantum datatypes) or averages (if flow datatypes).
    • Real Time: Since measurements are stored in real-time, no aggregation will take place.
    • Daily: Measurements between the date range will be aggregated per day.
    • Weekly: Measurements between the date range will be aggregated per week.
    • Monthly: Measurements between the date range will be aggregated per month.
    • Annually: Measurements between the date range will be aggregated per year.
  • Display Count yn: If the Merged Output is Table and if the Aggregate Level is Daily or longer, then you have the option of displaying the number of measurements that make up each aggregation.
  • Output Medium:
    • Table: A table is displayed in the bottom window.
    • Gracechart: A chart is created using the Grace Chart generator.
    • Easychart: A chart is created using the Easychart generator.
    • Transmit: A text file is created to transmit to your computer.
