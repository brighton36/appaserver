Annual Hydroperiod Algorithm

For each station:

  • For stage: Threshold = Ground Surface Elevation + Elevation Offset
  • For depth: Threshold = Elevation Offset
  • For each Year between Begin Year and End Year:
    • Begin Wetting Event = Date such that
      • Date >= January 1, Year and
      • Date <= December 31, Year and
      • Average stage for Day >= Threshold and
      • Average stage for yesterday < Threshold
    • End Wetting Event = Date such that
      • Date > Begin Wetting Event and
      • Date <= December 31, Year and
      • Averge stage for Day < Threshold
    • Wetting Event = Period of Days between Begin Wetting Event and End Wetting Event
    • Wetting Event Count = Number of Wetting Events in Year.
    • Discontinous = Count( days above threshold for Year )
    • Minimum = Number of days of the shortest Wetting Event.
    • Averge = Discontinous / Wetting Event Count
    • Maximum = Number of days of the longest Wetting Event.
