Output Measurement Sets

This is a set of output processes that display all the datatypes from a single station.

  • Data Sets Transmit: A text file is created to transmit to your computer.
  • Data Sets Table: A table is displayed in the bottom window.
  • Data Sets Easychart: An Easychart is displayed in a new window.
  • Data Sets Gracechart: A Gracechart is displayed in a new window.
  • Aggregate Level: Optionally select an aggregate level to calculate sum (if quantum datatype) or average (if flow datatype).
  • Validation Level:
    • Provisional: none of the measurements selected will have been validated yet.
    • Validated: only validated measurements will be selected.
    • Either: all measurements will be selected.
  • Plot For Station Check yn: This flag is used to assist technicians and hydrologists to perform their duties. It only applies to the Gracechart and Easychart.
    • If Yes or left blank: a datatype will appear if STATION_DATATYPE.plot_for_station_check_yn = 'y'.
    • If No:
      1. a datatype will appear if STATION_DATATYPE.validation_required_yn = 'y'.
      2. conductivity will be excluded, only salinity.
    • In either case: DATATYPE.units must be populated.
  • Begin and End Dates:
    • Leave both blank will report the entire period of record.
    • Enter a full Begin Date and End Date will report on that date range.
    • Enter only the year in Begin Date will report from January 1.
    • Enter only the year in End Date will report to December 31.
    • Enter only the year and month in Begin Date will report from the first day of that month.
    • Enter only the year and month in End Date will report to the last day of that month.
    • Enter a full Begin Date and leave the End Date blank will report on the single day.
    • Enter a only the year in Begin Date and leave the End Date blank will report on the entire year.
    • Enter a only the year and month in Begin Date and leave the End Date blank will report on the entire month.
    • Leave blank the Begin Date but populate the End Date will report from the beginning of record to the End Date.
